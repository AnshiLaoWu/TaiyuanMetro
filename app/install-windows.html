在Windows上安装俺是老吴·太原地铁应用程序

注：在Windows的应用程序中，需要按Alt+←或Alt+→键以前进/后退页面。

一、可执行文件 (.exe)

1. 在此页面下载适用系统版本的文件并打开

2. 在弹出的窗口中，单击“Next”并选择安装位置

3. 单击“Next”并等待安装完成

4. 单击“Next”并选择是否现在运行、是否创建桌面快捷方式

5. 即可正常使用

二、安装包 (.msi)

1. 在此页面下载适用系统版本的文件并打开

2. 在弹出的窗口中，单击“Next”并选择安装位置

3. 单击“Install”并等待安装完成

4. 选择是否现在运行，即可正常使用
